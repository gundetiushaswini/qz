Frequently Asked Questions

What is the purpose of this project?

This project is designed to provide an engaging EduVide for learning purposes. It includes various quizzes to test knowledge and improve learning outcomes.

How can I participate in the quizzes?

Simply sign up, log in, and navigate to the quiz section where you can start answering the quizzes.

How can I track my progress?

Your progress is automatically saved after each quiz, and you can track your performance from the dashboard.

How can I create new quizzes?

Only admins can create new quizzes. If you're an admin, you can access the admin panel and manage quizzes there.
